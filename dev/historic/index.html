Historic

Historical irradiance, weather and power data, from 2007 to 7 days ago at 1-2km and 5 minutes resolution. For more information see the API Docs. The Historic module has 2 methods:

EndpointAPI Docs
radiation_and_weatherlist of API parameters
rooftop_pv_powerlist of API parameters

Example

using Solcast: Historic

res = Historic.radiation_and_weather(
        -33.856784,
        151.215297,
        "2022-10-25T14:45:00.000Z";
        output_parameters=["air_temp"],
        duration="P1D"
    )

As a DataFrame.jl dataframe

df = Historic.to_dataframe(res)
